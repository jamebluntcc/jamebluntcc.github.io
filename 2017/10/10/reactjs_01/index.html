reactjs 初识

2017-10-10 • 分类: js • 标签: reactjs

reactjs

reactjs 是由鼎鼎大名的 facebook 在2013年开源的一个数据驱动视图的 javascript 库。在长时间使用 jQuery 后，实在觉得 jQuery 有时候会让开发变得比较繁琐，特别是涉及到大量用户交互的时候。所以希望找到一个新的框架来配合 jQuery 使用。在这样的想法下便很自然的接触到了 reactjs 一眼就被组件化的设计模式所吸引。其实现在很多框架都有涉及到组件化，但是个人认为它是做的最彻底的。reactjs 试图将html, css, js 真正的封装在一起, 因为 JSX 一切都是 javascript。

配置 reactjs 环境

reactjs 其实依赖于一个很大的技术栈，转译使用 Babel 模块打包工具使用 Webpack，定制 build 需要 grunt 或者 gulp 而且这些技术栈需要自己配置，为了能够快速开发 reactjs 的应用可以使用一个叫 create-react-app 的npm 发布的安装包进行配置开发环境。
