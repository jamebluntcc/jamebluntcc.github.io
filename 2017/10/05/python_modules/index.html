python 标准库使用

2017-10-05 • 分类: program • 标签: python

functools

functools 包括了一系列操作函数的工具，简单的说就是高阶函数的使用。

  • partial：重新定义函数的签名，可以将一些函数的参数固定起来，调用时可以减少函数的参数传入
  • wraps：将被封装函数的__name__, __module__,__doc__ 等复制到封装函数中，这样可以获取到正确的源函数对于属性

还记得自己在写 web app 的时候自己实现 login_required 的时候，自己第一次看到 wraps 的使用。

在这里使用 wraps 是为了拷贝源函数的属性，否则在使用过程中会出现一些意想不到的错误。

collections

collections 里包含了 5 个高性能的数据类型：

  • Counter：一个快速计算的计数器工具
  • deque：一个双端队列

可见，collections 中的 deque 拥有和 list 相同的数据结构，只是和一般的列表最大的不同就是支持 popleft 和 appendleft 。其实咋一看感觉 deque 和 redis 也类似。在 redis 中有 rpush rpop lrange 等。

  • defaultdict：简化处理不窜在键的场景

defaultdict 的参数就是值的默认类型，也支持自定义数据类型。对于不同的数据类型都有默认值，int 是 0。
如果需要自定义默认值可以：collections.defaultdict(lambda x=10:x)

  • OrderedDict：有序的字典数据结构
  • namedtuple：能创建出通过属性访问的元素内容的扩展元组。

关于 namedtuple 的使用，在数据库的操作上有很大的作用。
