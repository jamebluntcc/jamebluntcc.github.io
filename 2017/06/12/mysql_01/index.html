mysql 基本操作

登录mysql

  • -h:指定客户端所登录的mysql主机名；
  • -u:登录用户名；
  • -p:登录密码；

查看mysql基本信息

  • 查看全部数据库：show databases;

  • 使用数据库：use database;或者在登录使用mysql -D databasename -h hostname -u username -p;

  • 查看表格基本信息: describe table;

mysql的权限管理

基本操作

  • 创建数据库：create database 数据库名 [其它选项];

  • 创建数据库表：create table 表名称;

    对于较长的sql语句可以写在文本中，然后再导入到mysql中。mysql -D test -u root -p < students.sql

    • 插入：insert into tablename values(...); 插入部分数据:insert into students (name,sex,age) values('Tom','male',21);
    • 查询:select 列名称 from 表名称 [查询条件]; select age,name from students where sex='male'
    • 修改: update 表名称 set 列名称=新值 where 更新条件;
    • 删除: delete from 表名称 where 删除条件;

    • 创建表后的修改:

      • 添加列:alter table 表名 add 列名 列数据类型 [after 插入位置];
      • 修改列:alter table 表名 change 列名 列新名 列数据类型;
      • 删除列:alter table 表名 drop 列名;
      • 重命名:alter table 表名 rename 新表名;

  • 删除整张表:drop table 表名;

  • 删除数据库:drop database 数据库名;

mysql数据的导出与导入

  • mysql数据导入:mysql -D databasename -u root -p < xxx.sql 一般先在sql外建立table后再导入到相应的数据库;
  • mysql数据导出(数据表的结构和数据)mysqldump -u root -p databasename > xxx.sql
