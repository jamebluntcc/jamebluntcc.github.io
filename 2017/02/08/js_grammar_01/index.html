js的语法基础(一)

array

js’s array本质是object;

array的length属性，只要是array一定会有length属性。而且array的length属性是可写的，定义为该数组键名的最大整数值加1。所有清空一个数组的方法：arr = []或者arr.length = 0。由于数组是对象的一种所有可以添加数组的属性，但不会影响到length属性;

js中还存在着类数组对象，定义只要有length属性的对象就是类数组对象。但是length属性不是动态值。典型的类数组对象是函数的参数arguments对象，以及大部分的DOM元素集，还有字符串。

数组的slice方法将类数组对象，变为真正的数组。var arr = Array.prototype.slice.call(arrayLike);

js错误处理机制

Error对象

js在运行过程中出现错误就会抛出一个错误对象，js提供一个Error构造函数，所有抛出的错误都是这个构造函数的实例。

我们调用Error构造函数生成一个err实例，Error构造函数接收一个参数，表示错误提示,可以从实例的message属性读到这个参数。而且js规定Error对象的实例必须有message属性。 js中的6类错误(都是Error对象的派生):

  • SyntaxError:代码解析出错;
  • ReferenceError:引用一个不存在的变量;
  • RangeError:值超出范围;
  • TypeError:类型出错;
  • EvalError:eval函数没有被正确执行;
  • URIError:URI相关函数参数不正确;
