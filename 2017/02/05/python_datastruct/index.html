python的基本数据结构

对于任何一门计算机语言第一步都是要熟悉它的数据结构.

pyhton中的数据结构：

  • tutple
  • list
  • dict

repr 函数会创建一个字符串，以合法的python表达式的形式去表达值 str repr 反引号都是将python值转为字符串的三种方法。其中str最易于阅读。

在python中，方法是指一种与特有对象相关的函数

list的方法：

  • append 修改list不返回值
  • extend 修改list不返回值
  • count 不修改list返回值
  • reverse 修改list不返回值
  • pop 修改list返回pop出的元素
  • index 返回index的值
  • remove 按照list的值修改list
  • insert 按照list的位置插入值
  • sort 修改list不返回值

tips:

  1. python中复制list不是简单的 x = y,而是y = x[:] y = x 仅仅是将x指向y;

  2. 取得sort后的list一般可以用y = sorted(x)获得一个sort后的结果;

    高级排序

    sort方法总是对按照python默认的排序方法进行排序，如果我们需要按照自己的排序方法进行排序需要使用compare(x,y)的形式来自定义比较函数：

    sort还有两个可选择参数：key reverse： key接收一个函数，按照key产生的键来进行排序； reverse 在布尔值：True 或者 False;

string

字段宽度与精度

%10.3f表示按照右对齐10个字段，小数点保留3位，如果是 %-10.3f负号表示左对齐。%+10.3f表示无论数字是正还是负都要显示符号。

字符串的方法：

find:在一个字符串中找到一个字串，返回字串的起始下标，这个方法还可以接收起始和终止的下标：str.find('funny',1,10);

join:是split的逆向操作，将一个list对象进行连接； ','.join([1,2,3]);

replace:字符串替换方法'I\'m jamebluntcc'.replace('cc','xy')

split:join的逆向操作,用于将字符串进行分割返回一个list.'usr/bin/env/'.split('/')

字典的方法

  • clear 清空dict d.clear() d=={}

  • copy copy返回一个复制的dict，但是需要注意的是这个方法得到的dict替换值的时候原来的dict不会受到影响，如果要修改一个值时，它会原地修改，原始的dict也会变。所有为了避免这个问题要使用深度复制(deepcopy)

  • fromkeys formkeys使用已经给定的键值创建一个新dict默认所有的值都是None:

  • get 提供一个更加友好的方式去访问键值所对应的值d.get('name')如果不存在这个键值返回None

tips: dict：k in v 指判断键值是否在dict中 list：k in v 指value是否在list中

  • items 将dict返回为list，list中每一项为(key,value) iteritems返回迭代器
  • keys 以list的形式返回dict的全部key
  • values 以list的形式返回dict的全部value
  • pop pop方法获得对应键的值，然后将键-值移除
  • update 利用一个dict更新另一个dict
