2017-07-05 • 分类: program • 标签: python

关于字符串

python 字符串存在两种，一种是普通字符串 一种是 unicode 字符串。

  • python 字符串中可以在前面加上 R/r 表示是原生字符串;
  • python 字符串中可以在前面加上 U/u 表示是 unicode 字符串;
    其次 python 的字符串不能被改变，这意味着所有关于字符串的操作都是创建新的字符串。

字符串的常用技巧

  • strip():去掉字符串两端的空格或者其它字符,常用于读取文件或者处理文件目录上
  • join:join 方法不需要多说，但是需要注意一点的是当使用创建大字符串时使用 join 比 += 有优势很多。
  • 快速将一个字符串反转的方法:revese_str = str[::-1]

关键技巧: 总是在IO的时候进行转换;所有外来的文本进入程序时都需要先转换为 unicode 当数据从程序出去时都需要转为相应的编码字符。

首先 str 和 unicode 都是 basestring 的子类，str 严格的说应该是字节串，它是 unicode 经过编码后的类型

在进行IO时，使用 open() 方法打开文件 read() 使用读取的是 str 所以读取后要进行 decode() 在 write() 时如果是 unicode 就需要 encode。另外 模块 codecs 提供的 open 方法在打开文件时读取返回是 unicode。
